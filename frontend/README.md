# Frontend Trendsee

Frontend-часть проекта находится в папке `frontend/` и реализована на Vue 3 + Vite.

Запуск проекта рекомендуется выполнять из корня репозитория общей командой:

```bash
docker compose up --build
```

Основные директории:

- `src/components` — переиспользуемые Vue-компоненты
- `src/pages` — страницы приложения
- `src/router` — маршрутизация
- `src/services` — API-клиент и вспомогательная логика
- `src/styles` — глобальные стили

Подробное описание проекта, запуск backend, структура и Postman-коллекция находятся в корневом [README.md](../README.md).
