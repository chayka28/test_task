# Frontend Trendsee

Frontend-часть проекта находится в папке `frontend/` и реализована на Vue 3 + Vite.

## Что реализовано

- страница ленты публикаций;
- карточки с краткими данными публикации;
- модальное окно с полной информацией о публикации;
- infinite scroll;
- регистрация и вход;
- профиль пользователя;
- создание, редактирование и удаление публикаций через интерфейс;
- публикации с поддержкой видео, превью и ссылки на источник;
- раздел избранного;
- рабочая кнопка догрузки `Найти еще ролики`;
- раскрывающийся блок `Creative +`;
- визуальные элементы макета, которые пока не реализованы как отдельные модули, оставлены как заглушки с уведомлением.

## Запуск

Рекомендуемый способ запуска всего проекта:

```bash
docker compose up --build
```

Локальный запуск только frontend:

```bash
cd frontend
npm install
npm run dev
```

## Основные директории

- `src/components` — переиспользуемые Vue-компоненты
- `src/pages` — страницы приложения
- `src/router` — маршрутизация
- `src/services` — API-клиент, localStorage, форматирование
- `src/styles` — глобальные стили

Полное описание backend и инструкции по сдаче находятся в корневом [README.md](../README.md).
